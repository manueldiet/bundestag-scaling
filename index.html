Introduction

When a parliament votes, it leaves a trace. Each roll call is a small data point about where every legislator stands relative to every other. With enough such traces, one can recover an underlying geometry — a low-dimensional space in which familiar political categories reappear as coordinates.

This paper applies three such estimators to the namentliche Abstimmungen of the 20th German Bundestag. Crucially, we incorporate a legislative codebook that tracks whether each roll-call motion was ultimately accepted or rejected. As we will see, this binary indicator fundamentally changes how we interpret the recovered dimensions.

Data Pipeline & Imputation

The dataset comprises 161 roll-call votes across 772 legislators. We coded Yes as 1, No as 0. To handle missingness for matrix-based methods (SVD), we utilize a double-mean imputation strategy. This balances the individual MP's baseline tendency to vote "Yes" with the specific bill's overall popularity.

# Task 1.3 — Double-Mean Imputation (01_data_preparation.R)
double_mean_impute <- function(X) {
  row_means  <- rowMeans(X, na.rm = TRUE)
  col_means  <- colMeans(X, na.rm = TRUE)
  grand_mean <- mean(X, na.rm = TRUE)
  
  # outer(row_means, col_means, "+") produces the Matrix
  # row_means[i] + col_means[j] in one go
  imputed_values <- outer(row_means, col_means, "+") - grand_mean
  
  # Clip to [0, 1] bounds
  X_imp <- ifelse(is.na(X), imputed_values, X)
  X_imp <- pmin(pmax(X_imp, 0), 1)
  
  return(X_imp)
}

Baseline SVD

Singular Value Decomposition writes the imputed vote matrix as X = UΣVᵀ. The first column of U gives each legislator a coordinate on the dominant axis of variation.

This spatial clustering makes perfect sense from an institutional perspective: The model precisely separates the government coalition (SPD, Grüne, FDP clustered tightly together) from the opposition blocs.

However, this also reveals the methodological limits of using a raw SVD for ideological scaling: By forcing voting behavior onto a single mathematical dimension, the algorithm pushes the AfD and Die Linke extremely close together. Ideologically, these two parties have virtually no overlap. The SVD is fundamentally blind to why a party votes "No". It simply registers that both parties systematically reject government bills. Therefore, Dimension 1 is not measuring a traditional left-right political compass, but rather pure bloc behavior (approval vs. opposition of the government agenda).

b) Variance explained by Dimension 1

The proportion of variance explained confirms that one structural cleavage dominates everything else in this parliament.

Double-Centered SVD

To strip out baseline tendencies — some MPs vote yes more often; some bills pass more easily — we double-center the matrix before decomposing it. This isolates the pure polarization underlying the votes.

# Task 3.1 — Apply Double-Centering (03_double_centered_svd.R)
double_center <- function(X) {
  row_means  <- rowMeans(X)
  col_means  <- colMeans(X)
  grand_mean <- mean(X)

  # Efficient vectorised double-centering
  X_dc <- X -
    outer(row_means, rep(1, ncol(X))) -
    outer(rep(1, nrow(X)), col_means) +
    grand_mean

  return(X_dc)
}

Variance in Dimension 1 drops from 86.1% to 60.5%. The CDU/CSU emerges as the most extreme party, surpassing the AfD. Once we mathematically control for the baseline success rates of bills, the CDU/CSU's high volume of subsequently rejected alternative motions makes them the structural antipode of the government.

Bayesian 2-PL IRT

SVD produces no uncertainty estimates. We fit a two-parameter logistic IRT model via brms / Stan to extract proper likelihoods and posterior distributions, separating legislator positions ($\theta_i$) from vote difficulty ($\beta_j$) and discrimination ($\alpha_j$).

# Task 4.2 — Model Specification & Priors (04_irt_model.R)
irt_formula_2pl <- bf(
 response | trials(1) ~ exp(logalpha) * (theta - beta),
 theta    ~ 0 + (1 | person),
 beta     ~ 0 + (1 | item),
 logalpha ~ 0 + (1 | item),
 nl       = TRUE
)

irt_priors <- c(
 prior(normal(0, 1),   class = "sd", nlpar = "theta"),
 prior(normal(0, 2),   class = "sd", nlpar = "beta"),
 prior(normal(0, 0.5), class = "sd", nlpar = "logalpha")
)

Substantive Claim Evaluation

Claim: The FDP's position in the Ampel coalition was ideologically closer to CDU/CSU than to SPD/Grüne.

Rather than relying on point estimates, we draw jointly from the posterior distributions of all three party means and compute — for each MCMC sample — whether the FDP is closer to CDU/CSU or to SPD.

# Task 5.3 — Pairwise & Group Ordering Probabilities (05_substantive_evaluation.R)
# P(|FDP - CDU| < |FDP - SPD|) — "FDP closer to CDU than SPD"
dist_fdp_cdu <- abs(fdp_draws - cdu_draws)
dist_fdp_spd <- abs(fdp_draws - spd_draws)
p_closer_to_cdu <- mean(dist_fdp_cdu < dist_fdp_spd)

cat(sprintf("P(FDP closer to CDU/CSU than to SPD): %.3f\n", p_closer_to_cdu))
# Result: 0.000
Empirical Verdict The Bayesian posterior rejects the claim with P = 0.0%: in every single posterior draw, the FDP is closer to the SPD and Grüne than to the CDU/CSU. While public debates often highlighted policy friction, the formal roll-call data measures behavior constrained by the Koalitionsvertrag. Under strict coalition discipline, the FDP functioned as an integrated block within the governing majority.
