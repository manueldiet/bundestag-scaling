- `svd_interactive.html`     ← from `output/svd_interactive.html` of Phase 1
- `dc_svd_interactive.html`  ← from `output/dc_svd_interactive.html` of Phase 1

Also expected (when ready):
- `wp2_svd.RData`, `wp3_dc_svd.RData`, `wp4_irt.RData` — used by `scripts/06_export_for_web.R`

Static PNGs (`svd_scree.png`, etc.) belong in `../img/`.
